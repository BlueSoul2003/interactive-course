(A) Read the texts about three different housework. Then, assign each letter
to the statement that best represents it.
Setting the Table
Setting the table means to place plates, cups, forks, spoons and
napkins in the correct spots before a meal. It helps prepare the
space for eating. This task needs neatness and careful attention to
where each item should go. It's usually done before lunch or dinner
and can be done by children or adults.
Doing the Laundry
Doing the laundry includes sorting clothes, putting them into the
[8 ] washing machine, drying them and folding them after. This task
helps keep clothes clean and ready to wear. It may also include
ironing. It takes time and often needs more than one step.
Dusting the Shelves
Dusting the shelves is the task of wiping away dust and dirt from
shelves and surfaces. It helps keep the home clean and free from
dust that can build up over time. This job is usually done with a
cloth or duster and needs to be done gently so that nothing falls or
breaks.
