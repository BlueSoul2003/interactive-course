Static-friendly GitHub workflow

GitHub Pages cannot create folders from the public website because it has no server write access. This helper gives the folder path to create in GitHub, then the catalogue generator picks up the new PDFs.

  1. Create the suggested folder path in GitHub or locally.
  2. Upload the PDFs with clear names using underscores instead of spaces.
  3. Run python tools/generate_pdf_catalog.py before publishing, or let a GitHub Action run it later.
